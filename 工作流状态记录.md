# 工作流状态记录

> 状态：传播工作流接续资产  
> 主责：记录涛哥创作工作流每轮做到哪里、产出了哪些交接物、下一次如何恢复。  
> 边界：本文件不替代热点候选池、选题库、Brief、文案、画中画方案或交付记录；它只做 save / restore / report 的轻量状态索引。

---

## 使用规则

1. 每轮 workflow 暂停、完成、返工或等待涛哥选择时，更新一条 `workflow_session_record`。
2. 下一次用户说“接着上次 / 继续热点创作 / 回到 workflow”，先读本文件，再读对应交接物。
3. 不用本文件保存完整正文，只保存路径、ID、当前阶段和下一步建议。
4. 如果状态和具体交接物冲突，以交接物字段词典和实际产物为准，并修正本文件。
5. 如果用户确认放弃本条，`session_status` 写 `session_archived`，不要删除历史记录。
