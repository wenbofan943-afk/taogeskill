选题与切口

切口：画中画不是装饰，而是留存任务驱动的视觉资产。

目标：验证一张 required 图片能否从 visual beat 追溯到 prompt、generation record、image asset 和 HTML embed manifest。

正式文案

别急着给短视频加图，先问一句：这张图到底帮观众多停哪 2 秒？

如果一张图只是好看，但和这句话没关系，那它不是画中画，是干扰。

真正有用的画中画，应该能说清楚三件事：放在哪一句旁边，解决哪个留存风险，不用它会损失什么。

所以这套流程不是先问“出几张图”，而是先拆口播、找风险，再决定哪一张必须生成。
