## 1. R5 要解决什么

当前工作流可以从账号档案出发做内容，但热点发现仍依赖本轮临场调研，视觉一致性也主要停留在单篇内容。R5 要解决两个业务问题：

| 问题 | R5 的产品回答 |
|---|---|
| 多账号会不会做成同一套视觉、同一套选题？ | 工作流提供统一能力；每个账号持有独立视觉身份、栏目模板、雷达策略和私有记忆。 |
| 二手车账号会不会被泛汽车热榜带偏？ | 先筛二手车直接事件；不足 3 条事实可核验候选时才开启有证明的新车外溢。 |
| 热点是否只是“标题里有关键词”？ | 先把来源观察归并为事件，再判断账号相关性、受众收益与内容角度。 |
| 热度分能否代表事实？ | 不能。来源、事实、传播、风险、趋势和账号匹配分别记录。 |
| 词库是否会越跑越偏？ | 引擎可自由探索扩词，但以辅助发现后被选择的候选计数自动升降权；硬禁区与用户封禁仍优先。 |

R5 不追求“全网不漏任何热点”。首期成功标准是：对账号真正要讲的事件，能够说明**怎么发现、为什么与该账号有关、事实到什么程度、能服务哪类受众、为什么值得做成选题**。

## 2. 与 R1-R4 的关系

```text
R5 账号策略 / 视觉身份 / 雷达
  -> R1 选题卡、Brief、口播文案
  -> R3 单篇 visual_need_analysis 与图片资产
  -> R2 多账号或多选题时的隔离和 fan-out
  -> R4 公开样例只使用脱敏策略，绝不带真实账号记忆
```

R5 不替代：

```text
R1 的内容质量与最终交付链。
R2 的 session、分支与恢复模型。
R3 的单篇 0..N 视觉需求分析与图片资产链。
R4 的公开包净化和发布治理。
```

## 3. 工作流能力与账号资产的分工

| 工作流共用能力 | 账号独立资产 |
|---|---|
| 来源快照、信号登记、事件归并、证据状态、趋势比较、候选排序、关键词候选生成 | `account_topic_policy`、`query_lexicon`、排除词、`hotspot_memory`、`visual_identity`、栏目模板、受众优先级 |
| 同一事件可被多个账号评估 | 每个账号各自产生 `account_fit_reason`、主受众、外溢证明和选题角度 |
| 视觉规划器读取身份合同 | 单篇图片数量仍由 R3 内容视觉需求分析产生，不能由账号固定配额 |

运行时不把散文账号介绍直接塞给雷达。账号档案通过 `radar_policy_ref` 与 `visual_identity_ref` 传入版本化结构；缺少引用时应诚实降级为 `account_policy_incomplete`，不得假装按账号策略筛选过。多账号时，账号档案还是唯一身份绑定真源：`account_display_name`（人看）、私有目录键（路径）和 `account_technical_slug`（机器）分工明确，策略、词库、视觉资产和 session 快照必须与同一 `account_identity_id` 绑定；任一错绑或跨根引用均为 `account_identity_inconsistent`，不能静默回退。

在进入热点、选题或内容 Skill 前，运行时还必须执行账号启动检查。它只补当前任务需要的缺口，并把用户确认写成版本化的 session 账号快照；下游读取同一快照而非再次解释散文档案。热点发现不因缺视觉身份而阻断，视觉交付才按需要求视觉身份。高风险话题默认只做交叉核验和行业机制拆解，不直接对具体主体下结论。

## 4. 用户可感知的业务流程

```text
选择账号
  -> 账号启动检查；只补当前任务相关的缺口并冻结 session 账号快照
  -> 读取账号策略、视觉身份与批准词库
  -> 发现来源信号（含人工监看，但必须标记）
  -> 归并为事件、记录来源/事实/传播/风险
  -> 按账号判断二手车直接性与主受众
  -> 形成候选池；必要时有证明地开启新车外溢
  -> 用户选题
  -> R1 内容生产；R3 按账号身份辅助视觉规划
```

对用户而言，看到的不是“算法说 92 分”，而是一张可判断的候选卡：

```text
事件是什么；从哪里发现；事实到什么程度；现在是新发现还是确有升温；
它为何影响二手车；主要服务买车者、卖车者还是车商；可讲什么、不该怎么讲。
```

## 5. 人类门禁与风险边界

| 门禁 | 为什么必须由人判断 |
|---|---|
| 账号策略首次启用 / 改版 | 决定账号长期定位与禁区，不能由一次热点反推。 |
| 词库硬封禁 / 策略改版 | 扩词不逐词审批；但禁区、合规边界和账号长期方向仍需由人维护。 |
| 高风险事件进入选题 | 涉及名誉、金融、消费权益等时，媒体报道不等于最终定论。 |
| 新车外溢进入候选 | 必须确认其对二手车有真实、可说明的传导关系。 |
| 视觉身份实际资产 | 色彩、字形、角标、Logo 等需用户确认，不虚构品牌资产。 |
