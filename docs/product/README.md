## 当前产品优先级检查点

本项目仍是自用的内容生产 workflow，不是内容 SaaS，也不是自动发布平台。核心交付是把账号 / 产品 / 热点输入稳定走到人可验收、可继续修改、可由人决定发布的完整内容交付包。

下一阶段优先以少量真实账号、真实选题和真实交付重复验证两条已编译主路径：`直供创作`、`热点创作`。人保留选题判断与最终发布判断；每次真实运行记录是否完成交付、人工返工点、断点恢复和实际使用感受。后续结构投资只由这些真实压力触发。

未来的最终承载面是“短视频创作项目”，面向运营协作，其对象关系为 `IP -> 方向组/业务 -> 账号 -> 作品`。本工作流当前独立运行，只为验证可稳定交付的内容生产引擎，不另建平行 SaaS。未来仅预留一个能力槽位：工作台提供 IP / 账号 / 作品上下文与创作意图；本 workflow 返回可追溯内容交付包、质量结果、人工确认状态与可继续修改的草稿。当前不实施 API、数据库、前端接入，不复制该项目对象模型，也不预建平台化路径；真实运行结果决定后续接入合同。

公开发布 / 兼容链路在当前必要修复完成后冻结；没有来自真实交付的直接阻断，不新增平台分支、编排层、自动发布或泛化 SaaS 能力。
