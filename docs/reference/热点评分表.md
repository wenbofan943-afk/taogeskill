# 热点评分表

> 状态：当前传播研究工作表
> 主责：按 12 分规则评价热点与账号母题、产品能力、目标人群的关联度。
> 边界：评分通过只代表可进入选题，不代表最终文案可发布。

---

## 评分规则

每项 0-2 分，总分 12 分：

| 维度 | 0 分 | 1 分 | 2 分 |
|---|---|---|---|
| 人群相关 | 和目标用户无关 | 间接相关 | 直接相关 |
| 母题相关 | 不属于任何账号母题 | 可勉强接入一个母题 | 天然属于一个母题 |
| 产品相关 | 产品接不上 | 只能间接提到产品 | 能自然展示产品能力 |
| 情绪强度 | 无明显情绪 | 有讨论但不强 | 有明显痛点、焦虑、反差或共鸣 |
| 风险可控 | 高风险，不建议做 | 有风险，需要克制表达 | 风险低，可正常做 |
| 可落地性 | 只能聊观点 | 能做一篇内容 | 能做系列内容或组合内容 |

结论：

```text
0-4 分：淘汰
5-7 分：只做轻观点或评论
8-10 分：进入正式选题
11-12 分：重点热点
```

## 桥接链门槛

评分前必须先判断桥接链：

```text
S 池热点：S -> B -> A -> C -> D
B 池热点：B -> A -> C -> D
A/C 池热点：A -> C -> D
```

如果关键跳转只能靠想象，即使单项评分看起来高，也不能进入正式选题。

桥接状态：

```text
未桥接：没有写出链路，不能进入选题。
弱桥接：有链路，但关键跳转依据薄，只能待补桥或降级轻观点。
可桥接：链路基本成立，可以进入评分。
强桥接：每一跳都有清楚依据，优先推荐。
硬蹭嫌疑：看似有链路，但主要靠口号、类比或情绪硬拉，淘汰或重写角度。
```
