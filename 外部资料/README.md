## 一、使用原则

1. 外部资料只用于学习成熟项目的结构、方法和提示词工程。
2. 不把外部资料直接复制成本项目规则；必须结合涛哥创作工作流重新抽象。
3. 引用外部项目思路时，要保留来源说明和必要鸣谢。
4. 如果外部资料是截断副本、部分文件或示例目录，必须视为“不完整资料”，不要按完整仓库使用。
5. 不在本目录保存账号密码、API Key、真实客户资料、平台 Cookie 或不可公开数据。
