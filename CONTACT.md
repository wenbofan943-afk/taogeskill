# Contact

This project is maintained as an alpha content workflow / skill kit.

## Feedback

Use GitHub Issues for public-safe feedback:

- workflow usage problems
- sample run problems
- feature requests
- unclear documentation
- release package issues

Please do not include private account files, customer records, platform cookies, API keys, tokens, phone numbers, WeChat IDs from other people, or unpublished production content in public issues.

## Security And Privacy

For security, privacy, secret, or unsafe automation reports, read `SECURITY.md` first.

If a report contains sensitive values, remove the value and describe the location pattern instead. Do not publish real secrets or private account data in public issues.
