# Code of Conduct

Use this workflow to improve content operations, not to harass, impersonate, spam, or automate platform interaction.

Contributions should keep the project readable for both humans and AI agents.

## Collaboration Standard

```text
Be specific.
Keep feedback tied to artifacts, fields, samples, or skills.
Do not include private account data or platform credentials in public issues.
Respect the project boundary: content workflow, not platform automation.
```
